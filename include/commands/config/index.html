jtl config init#

$ jtl config init [OPTIONS]

Создать (перезаписать) конфигурационный файл.

Параметры#

CLI Prompt Config Описание
--local/--global - - Локальный (./) или глобальный (~/) конфиг
-js/--server + server Cервер Jira
-jl/--login + login Логин в Jira
-jp/--password + password Пароль в Jira
-p/--project + project Ключ проекта по умолчанию
-tl/--template - issue_template YAML-файл с шаблоном полей задачи
--open/--no-open + open_in_browser Открывать созданные задачи в браузере

Примечание

Пропущенные обязательные параметры командной строки будут предложены для ввода после запуска команды
